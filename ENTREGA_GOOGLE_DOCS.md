# Entrega final - Atelier YaninaCattadori

## Codigo

El proyecto esta construido como una web estatica institucional de 5 paginas: `index.html`, `servicios.html`, `alta-costura.html`, `galeria.html` y `contacto.html`. Todas las paginas tienen estructura semantica, navegacion completa, metadatos SEO, enlaces a Bootstrap, Bootstrap Icons, AOS y estilos propios compilados desde SASS.

Capturas sugeridas:

- HTML de `index.html` mostrando `head`, nav y estructura semantica.
- SASS de `assets/scss/abstracts/_variables.scss`.
- SASS de `assets/scss/abstracts/_mixins.scss`.
- SASS de algun componente, por ejemplo `assets/scss/components/_cards.scss`.

## Reflexion final

La tematica elegida fue un atelier de arreglos de ropa y alta costura. La idea fue presentar un servicio cotidiano con una identidad elegante, artesanal y profesional, mostrando tanto arreglos practicos como proyectos de diseno a medida. El sitio busca que una persona pueda conocer los servicios, ver trabajos, entender el proceso y pedir turno desde cualquier pagina.

Una dificultad importante fue adaptar Bootstrap a una identidad propia para que el sitio no quedara generico. Tambien fue necesario cuidar el responsive, especialmente el menu mobile y las animaciones laterales, para evitar desbordes horizontales. La solucion fue organizar los estilos con SASS, crear componentes reutilizables y agregar pequenos fallbacks en JavaScript para que la navegacion siga funcionando aunque una libreria CDN no cargue.
